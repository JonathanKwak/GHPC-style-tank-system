# GHPC-style-tank-system
I like the game Gunner, Heat, PC! and I decided to recreate its feel onto Roblox.

Features:
- multiplayer compatibility for commander, driver, and gunner.
- Ability to enter and exit views for a more immersive experience
- For gunner; Loading various ammo types, ejecting the current round, night vision, turret traversing, zooming, projectile simulation and replication across different clients
- For commander; full 360 degree view, compass system, night vision, zooming
- For driver; engine toggling, realistic physics simulation for tracks and wheels, gear switching

Note: This system uses external modules like FastCastRedux (by Eti the Spirit) and TurretController (by Dthecoolest)

Additionally, the github only has the code uploaded, none of the objects themselves (i.e, UI, models, joints, etc...)
The scripts are mainly server-scripts, although their RunContexts are based on their names. So for instance, "GunnerClient" is set to RunContext of Client.
The entire system (all self-contained) is attached in the Tank.rbxl file.
